# life-plan-lab

FIRE・住宅ローン・教育費・老後を統合した生活設計シミュレーター（V2）。

「未来を当てる」ためではなく「未来を整理する」ためのツール。入力して終わりではなく、
**結果を見る → 条件を変える → また結果を見る → 家族で話す** という試行錯誤を支えることを重視しています。

## 開発

```bash
npm install
npm run dev      # 開発サーバ（http://localhost:5173）
npm run build    # 型チェック + 本番ビルド（dist/）
npm test         # ロジック/ストア/描画のテスト（vitest）
```

静的アプリとしてビルドし、将来 `/lifeplan-v2/` などのサブディレクトリに独立ページとして
公開する想定（`vite.config.ts` の `base: './'` で相対パス出力）。

## 実機・スマホでの確認手順

1. `npm install`
2. `npm run dev`
3. 表示された URL（既定 `http://localhost:5173`）をブラウザで開く
4. Chrome DevTools のデバイスツールバー、またはスマホ実機で **幅375px（iPhone相当）** にする
5. ざっくり診断を最初から最後まで操作する
6. 結果画面と各 Bottom Sheet を開いて確認する

**スマホ実機（同じ Wi-Fi）で見る場合：**

```bash
npm run dev -- --host 0.0.0.0
```

起動後に表示される **Network URL**（例 `http://192.168.x.x:5173`）をスマホのブラウザで開く。

**開発用ショートカット：** モード選択画面の下に「サンプルで結果を見る（開発用）」リンクがあります
（`npm run dev` 時のみ表示）。9問を手入力せずに結果画面・Bottom Sheet を確認できます。

### 確認時にスクショを撮るとよい画面

次の修正指示を出しやすくするため、以下を 375px 幅で撮っておくと便利です。

- モード選択
- ざっくり診断の各ステップ（5枚）
- 結果 Hero ＋ 今回のポイント
- 人生タイムライン要約
- 資産推移グラフ（通常表示）
- グラフ拡大 Bottom Sheet
- 試算条件の折りたたみを開いた状態

## アーキテクチャ

- `src/schema` … 型・入力スキーマ（`Field<T>{value, source, label, assumptionText}`）・標準値/おすすめ値・正規化（両モードの合流点 `buildFullInput`）・ざっくり診断の質問定義
- `src/engine` … 純粋関数の計算群（年次/教育費/住宅ローン/FIRE/判定）。UIから完全分離。定数は `engine/constants.ts`
- `src/store` … zustand。ステップ遷移・入力状態・自動保存（localStorage）
- `src/features` … 画面（mode-select / input-steps / results / resume）

## 実機確認チェックリスト

**幅375px** で、各画面を確認してください。

### 画面ごと

- **モード選択**：ざっくり/しっかりが選べる
- **ざっくり診断：あなたについて**：年齢・年収・資産。Help/Skip が圧迫しない
- **ざっくり診断：お子さま・教育**：人数選択。0人なら教育方針が非表示
- **ざっくり診断：住まい**：持ち家/賃貸/購入検討
- **ざっくり診断：これからの働き方**：働き方＋仕事を減らす年齢（Recommended/Skip）
- **ざっくり診断：投資のスタイル**：安定/バランス/成長（Recommended/Skip）
- **下部固定ナビ**：常に見える／本文と重ならない／押しやすい
- **Help（？）**：開閉できる
- **Skip / Recommended**：押すと状態が分かる（✓表示）
- **未回答時の「このまま進む」**：赤字エラーにならず案内が出る
- **localStorage 再開オーバーレイ**：入力途中でリロードすると「続きから再開しますか？」
- **結果 Hero**：総合結果・FIRE達成度がすぐ分かる／大きすぎない
- **今回のポイント**：最大3つで分かりやすい
- **人生タイムライン要約**：主な節目が最大5件で長すぎない
- **資産推移グラフ（通常）**：横軸が年齢／はみ出さない／主張しすぎない
- **グラフ拡大 Bottom Sheet**：ツールチップ（◯歳（西暦）／資産）・節目マーカー
- **タイムライン詳細 Bottom Sheet**：全イベント（年齢・西暦・説明）
- **教育費詳細 Bottom Sheet**：教育費が増える年が分かる
- **住宅ローン詳細 Bottom Sheet**：未入力時は「しっかり診断で設定」を案内
- **条件を変えてみる導線**：各カテゴリへ戻って再計算できる
- **しっかり診断で詳しく見る導線**：入力を引き継いで遷移
- **今回の試算条件（折りたたみ）**：開くと利回り・インフレ・税制簡略化など

### 確認観点

- スマホ1画面に収まりやすいか／無駄なスクロールが少ないか
- 下部固定ナビが邪魔でないか／次へボタンが見つけやすいか／ボタンが押しやすいか
- Help / Skip / Recommended が圧迫していないか
- 入力中に不安になる表現がないか
- 結果画面の結論がすぐ分かるか／Hero が大きすぎないか
- 今回のポイントが分かりやすいか／タイムラインが長すぎないか／グラフが主張しすぎないか
- Bottom Sheet が開閉しやすいか／内部でスクロールしやすいか
- 試算条件が重すぎないか
- 全体が金融営業サイトっぽくなっていないか／静かで上質な印象があるか
